# Cartografía de lo Infinito

Un universo procedural, determinista e infinito generado mediante algoritmos cuánticos y ruido Simplex.

## Características
- **Exploración Seamless**: De las estrellas a la superficie de los planetas sin pantallas de carga.
- **Generación Determinista**: Cada coordenada del universo siempre genera el mismo contenido para todos los exploradores.
- **Sistema de Minería**: Extrae recursos de los planetas para mejorar tu nave.
- **Códex Universal**: Registra tus descubrimientos y compite en el ranking global.

## Desarrollo Local
1. Instala las dependencias: `npm install`
2. Compila el motor WASM (Rust): `cd procedural_engine && wasm-pack build --target web`
3. Inicia el servidor de desarrollo: `npm run dev`

## Tecnologías
- **Three.js**: Renderizado 3D de alta performance.
- **Rust/Wasm**: Motor de generación procedural optimizado.
- **GSAP**: Animaciones cinemáticas de interfaz.
- **Tone.js**: Paisajes sonoros generativos.
